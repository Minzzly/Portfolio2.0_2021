Refonte Portfolio

Description du projet :

Mon portfolio, au début, était un projet scolaire. J'ai créé un design et je l'ai codé au complet. Par contre, je n'étais pas aussi fière du résultat final, car le design était trop compliqué et n'était pas aussi plaisant à naviguer que je l'aurais pensé. Donc, j'ai décidé de faire une refonte complète de mon site.

J'ai fait :

  • Design du site
  • Le design du cv
  • L'html, css et js
  • Création des "thumbnails"
  • Le text et correction

Étape 1: Design

Le début de ma refonte :

Pour le design, je n'avais pas autant de temps de faire un design complet comme le premier. J'ai donc mis la couleur, quelques images et du texte générique. Mon but était de faire la base pour que je commence mon code le plus vite possible.

Étape 2: Programmation

L'html, Css et Js :

La plupart de l'html était déjà fait, donc il ne restait qu'a l'adapter pour le css. Beaucoup de mon temps a été dans le style, car j'ai dû regarder tous les fichiers pour ensuite les modifier ou les supprimer pour avoir mon resultat final. Pour le JavaScript, je n'ai pas modifier beaucoup de code a part le Loader et le Carousel.
